# User Management System

Ứng dụng quản lý người dùng đơn giản sử dụng React + Vite, tích hợp với JSONPlaceholder API.

## Tính năng

- Hiển thị danh sách người dùng với phân trang
- Thêm người dùng mới
- Chỉnh sửa thông tin người dùng
- Xóa người dùng
- Tìm kiếm người dùng theo tên

## Công nghệ

- React 19
- Vite
- JSONPlaceholder API

## Chạy dự án

```bash
npm install
npm run dev
```
